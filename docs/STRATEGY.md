## 1. 사용자가 얻는 가치

(아직 작성되지 않음)

> 작성 시점: 새 사용자 세그먼트가 추가되거나, 기존 사용자에게 새 가치 제안을 추가할 때.

## 2. 핵심 사용자 시나리오

(아직 작성되지 않음)

> 예: "Claude/Codex 사용자가 멀티 세션을 병행할 때 어떤 세션이 진행 중인지 한눈에 보여주고, 완료된 세션을 클릭 한 번으로 전환할 수 있게 한다."

## 3. 비범위 (Non-Goals)

(아직 작성되지 않음)

> 예: "범용 AI assistant 통합 (Cursor, Continue 등)은 v1 범위 아님 — Claude/Codex 전용 유지."

## 4. 성공 지표

(아직 작성되지 않음)

> 예: "주 7일 이상 메뉴바 상시 켜둠 사용자가 N명 이상."

## 5. 트레이드오프 / 제약

(아직 작성되지 않음)

> 예: "macOS only — Windows/Linux 대응은 비범위. Accessibility 권한이 필요한 동작은 fallback 경로 필수."
